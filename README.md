# coupon-process-plugin

飞书多维表格应用首页使用的「券策智能审核流程」静态插件组件。

## 内容

- 1 提交申请
- 2 智能判断
- 3 查询结果

第二步「智能判断」使用青绿色高亮，并加入轻量 CSS 动效。

## 本地预览

直接打开 `index.html`，或在项目目录启动任意静态服务。

## GitHub Pages

该项目无需构建工具，可直接部署到 GitHub Pages：

- Source：Deploy from a branch
- Branch：main
- Directory：/root

## 飞书嵌入建议

- 推荐插件容器高度：90–110px
- 页面为纯静态 HTML/CSS，无后端、无框架、无外部 CDN
- 如飞书容器高度较窄，当前版本已按单行流程条优化
